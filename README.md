Setup & Installation
# 1. Navigate to the blog app directory
cd blog-app

# 2. Install dependencies
npm install

Running the Project
# Start the development server (runs on http://localhost:5000)
npm run dev

The dev server will start with:

Hot Module Replacement (HMR) - changes auto-reload
Local URL: http://localhost:5000/
Press h + enter in terminal to see help options
Other Useful Commands
# Build for production
npm run build

# Preview the production build locally
npm run preview

# Run ESLint checks
npm run lint

# Build CSS (manual Tailwind build)
npm run build:css
